# ReadMe
Welcome to my Game Design Portfolio.
I specialize in gameplay mechanics, AI behaviour, economy loops, and level design, using Unity as my primary engine.
This portfolio showcases the three complete games I've designed and developed from scratch.


1. Zombie Hunters

Genre: Third-Person Shooter (TPS)
Platform: Mobile / PC
Engine: Unity

Overview:
Zombie Hunters is a fast-paced TPS where players take down waves of zombies in timer-based levels. Focus is on responsive controls, satisfying combat feedback, and scalable enemy AI.

Key Features:
a)TPS player controller with movement, aiming, shooting
b) Impactful visual/audio feedback (particles, muzzle flashes, SFX)
c) AI zombies with walking, chasing, and attack behaviours
d) Timer-based levels with increasing difficulty
e) Level design that allows expansions




2. Stronghold Defense

Genre: Tower Defense / Auto Battler / Strategy
Platform: PC / Mobile
Engine: Unity

Overview:
A hybrid strategy game mixing tower defense with auto battler mechanics. Features scalable enemy waves, resource economy, and deep unit upgrade systems.

Key Features:
a) AI for both player and enemy units
b) Tile-based placement logic with snapping
c) Unit upgrades with scalable cost curves
d) Gold-based economy forming the core gameplay loop
e) Endless-level possibility via config-driven wave generation
f) Balanced difficulty progression requiring strategic planning




3. LoreMaster

Genre: Trivia / Quiz
Platform: Mobile
Engine: Unity

Overview:
LoreMaster is a geek-culture trivia game featuring progression tiers, unlockable ranks, and multiple play modes.

Key Features:
a) Dynamic UI with smooth panel transitions
b) Randomized question & answer systems
c) Timer pressure, SFX, and screen-tint effects
d) Gold → Rank Up → Harder Questions progression system
e) Monetization-ready design structure

Additional game modes:
Mixed Category Mode
Eternal Mode (survive as long as you can)
